# Chrome Extension Developer - Vector Database Integration

We're seeking an experienced full-stack developer to create a Chrome browser extension that captures webpage content and integrates with a vector database via Ollama API, similar to the AnythingLLM Browser Extension.

## Required Skills:

### Technical Skills:
- **JavaScript (ES6+)**: Proficiency in modern JavaScript, asynchronous programming, promises, and async/await patterns
- **Python**: Strong programming skills with experience in web development and API integration
- **Browser Extension Development**: Extensive experience developing Chrome extensions, including:
  - Chrome Extension APIs and manifest files (v2 and v3)
  - Content scripts, background scripts, and popup UI development
  - Cross-origin resource sharing (CORS) and web security concepts
  - Browser storage mechanisms and local data persistence
  - Chrome Web Store publishing process

### Specialized Requirements:
- Develop a Chrome extension that extracts and processes current webpage HTML content
- Implement HTML content cleaning and preprocessing for vectorization
- Integrate with Ollama API to convert text content into vector embeddings
- Design and implement storage solution for vectors in a vector database
- Create intuitive UI elements for users to trigger content capture and search
- Implement secure API communication between the extension and Ollama service
- Optimize for performance and handle large HTML content efficiently
- Ensure proper error handling and user feedback mechanisms

### Preferred Experience:
- Experience with or ability to develop functionality similar to the AnythingLLM Browser Extension
- Understanding of how AnythingLLM extensions capture webpage content and send it to vector databases
- Familiarity with browser-based document scraping and content extraction techniques
- Knowledge of how to integrate with vector databases and AI services for semantic search
- Understanding of privacy and security considerations when capturing and transmitting webpage content
- Experience with RESTful APIs and web services
- Knowledge of data serialization formats (JSON, XML)
- Familiarity with version control systems (Git)

## Responsibilities:
- Design and develop a Chrome extension that captures webpage content
- Implement integration with Ollama API for vector embedding generation
- Create a robust vector database storage solution
- Ensure the extension is performant, secure, and user-friendly
- Follow Chrome extension best practices and security guidelines
